# Study Cards

## Description

Study Cards is a lightweight web application that presents learning content in a card based interface. The project is designed to support active recall and quick review by breaking information into small, focused units.

## Features

- Card based content display  
- Minimal, distraction free UI  
- Sequential navigation through study cards  
- Responsive layout for desktop and mobile  

## Use Cases

- Concept revision  
- Memorization and spaced review  
- Exam preparation  
- Self paced learning modules  

## Getting Started

### Prerequisites

- Modern web browser  
- No build tools or dependencies required  

### Usage

1. Open the site in a browser  
2. Navigate through cards one at a time  
3. Repeat sessions as needed for retention  

## Project Structure

- Static web page  
- Card components rendered via HTML and CSS  
- Optional JavaScript for navigation or interaction  

## Customization

- Modify card content directly in the source files  
- Add or remove cards as needed  
- Adjust styling to match different subjects or themes  

## Deployment

- Hosted on Hostinger  
- Delivered as a static site  
- SSL enabled  

## Scope

This project intentionally avoids complex features to remain fast, readable, and easy to maintain.
